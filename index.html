HELLO EVERYONE!!

Gosh, this is my longest streak in coding so far. I honestly started learning in 2017 but I have been on a long hiatus because I prioritized the wrong things in life. Jeez! It can be challenging but worth it.It feels so good to be back on track. This is the start of my journey into web development.
Here's one quote I found on the web that may help motivate you if you're also a beginner like me. Just keep going.
